RÉSUMÉ DES MODIFICATIONS

1. AJOUT DES TRADUCTIONS MANQUANTES
   - Ajout de toutes les clés de traduction pour C++, C# et Dart
   - Traductions complètes en français et en anglais

   - lang/fr/app.php : Ajout des traductions françaises
   - lang/en/app.php : Ajout des traductions anglaises

CLÉS DE TRADUCTION AJOUTÉES
================================================================================

Pour chaque langage (cpp, csharp, dart), les clés suivantes ont été ajoutées :

- why_learn_text : Texte d'introduction pour "Pourquoi apprendre"
- prerequisites_title : Titre de la section prérequis
- prerequisites_text : Description des prérequis
- prerequisites_items : Liste des prérequis
- prerequisites_note : Note importante sur les prérequis
- use_cases_title : Titre de la section cas d'utilisation
- use_cases_text : Description des cas d'utilisation
- use_cases_items : Liste des cas d'utilisation
- syntax_title : Titre de la section syntaxe
- syntax_text : Description de la syntaxe
- variables_title : Titre de la section variables
- variables_text : Description des variables
- datatypes_title : Titre de la section types de données
- datatypes_text : Description des types de données
- operators_title : Titre de la section opérateurs
- operators_text : Description des opérateurs
- conditions_title : Titre de la section conditions
- conditions_text : Description des conditions
- loops_title : Titre de la section boucles
- loops_text : Description des boucles
- functions_title / methods_title : Titre de la section fonctions/méthodes
- functions_text / methods_text : Description des fonctions/méthodes
- classes_title : Titre de la section classes
- classes_text : Description des classes
- collections_title : Titre de la section collections (C# uniquement)
- collections_text : Description des collections (C# uniquement)
- inheritance_title : Titre de la section héritage
- inheritance_text : Description de l'héritage
- interfaces_title : Titre de la section interfaces (C# uniquement)
- interfaces_text : Description des interfaces (C# uniquement)
- linq_title : Titre de la section LINQ (C# uniquement)
- linq_text : Description de LINQ (C# uniquement)
- async_title : Titre de la section async (C# uniquement)
- async_text : Description de async (C# uniquement)
- mixins_title : Titre de la section mixins (Dart uniquement)
- mixins_text : Description des mixins (Dart uniquement)
- futures_title : Titre de la section futures (Dart uniquement)
- futures_text : Description des futures (Dart uniquement)
- streams_title : Titre de la section streams (Dart uniquement)
- streams_text : Description des streams (Dart uniquement)
- flutter_title : Titre de la section Flutter (Dart uniquement)
- flutter_text : Description de Flutter (Dart uniquement)
- next_steps_title : Titre de la section prochaines étapes
- next_steps_text : Description des prochaines étapes
- next_steps_learned_title : Titre de la liste "Ce que vous avez appris"
- next_steps_learned_items : Liste des éléments appris
- next_steps_further_title : Titre de la liste "Pour aller plus loin"
- next_steps_further_items : Liste des éléments pour aller plus loin
