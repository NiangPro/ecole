\App\Http\Controllers MessageController

Contrleur pour la gestion de la messagerie interne

Ce contrleur gre toutes les fonctionnalits de messagerie :

  • Liste et affichage des conversations
  • Cration et envoi de messages
  • Gestion des pices jointes
  • Marquage des messages comme lus
  • Suppression de conversations et messages
