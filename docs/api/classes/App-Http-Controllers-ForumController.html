\App\Http\Controllers ForumController

Contrleur pour la gestion du forum communautaire

Ce contrleur gre toutes les fonctionnalits du forum :

  • Affichage des catgories et topics
  • Cration et gestion des topics
  • Systme de rponses et votes
  • Recherche dans le forum
  • Actions administratives (pingler, verrouiller)
