STRUCTURE DE LA SIDEBAR
------------------------

La sidebar contient deux cartes :

1. **Carte Profil** :
   - Avatar avec initiale
   - Nom et email
   - Statistiques (Membre depuis, Formations, En cours)

2. **Carte Navigation** :
   - Menu de navigation avec 6 sections
   - Icônes Font Awesome
   - États actifs avec gradient
   - Responsive avec toggle mobile

SECTIONS DISPONIBLES
--------------------

1. **Vue d'ensemble** (#overview)
   - Section d'accueil avec description
   - Statistiques générales

2. **Formations** (#formations)
   - Liste des formations en cours
   - Progression par formation

3. **Graphiques** (#charts)
   - Graphiques Chart.js
   - Activité sur 30 jours
   - Répartition des activités
   - Progression par formation
   - Scores des quiz

4. **Recommandations** (#recommendations)
   - Suggestions personnalisées
   - Formations recommandées

5. **Activités** (#activity)
   - Historique des activités récentes
   - Timeline des actions

6. **Objectifs** (#goals)
   - Liste des objectifs
   - Création et gestion d'objectifs

CSS AJOUTÉ
----------

- `.profile-sidebar` : Container principal
- `.profile-sidebar-card` : Cartes de la sidebar
- `.profile-sidebar-nav` : Navigation
- `.profile-sidebar-nav-link` : Liens de navigation
- `.profile-sidebar-nav-link.active` : État actif
- `.profile-sidebar-mobile-toggle` : Bouton toggle mobile

JAVASCRIPT AJOUTÉ
-----------------

- `scrollToSection(sectionId, event)` : Navigation vers une section
- `toggleSidebar()` : Toggle du menu mobile
- Détection automatique de la section active au scroll

UTILISATION
-----------

La sidebar est automatiquement affichée sur la page /profile.

Sur desktop :
- Sidebar fixe à gauche
- Navigation par clic ou scroll

Sur mobile :
- Bouton "Menu" pour afficher/masquer
- Navigation identique
