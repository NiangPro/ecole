DESCRIPTION DU PROJET
---------------------
Plateforme de formation gratuite en développement web avec système de badges,
certificats, quiz et exercices interactifs.

FONCTIONNALITÉS PRINCIPALES
----------------------------
- 15 Formations complètes (HTML5, CSS3, JavaScript, PHP, Bootstrap, Python,
  Java, SQL, C, Git, WordPress, IA, C++, C#, Dart)
- Système d'exercices interactifs avec éditeur de code intégré
- Système de quiz avec résultats détaillés
- Dashboard utilisateur complet avec statistiques
- Système de badges et certificats téléchargeables
- Panel admin sécurisé
- Gestion d'articles d'emploi par catégories
- Système de commentaires avec modération
- Newsletter automatique
- Système de favoris
- Notifications en temps réel
- Objectifs personnalisés
- Mode sombre/clair
- PWA installable
- Multilingue (FR/EN)
- SEO optimisé
- Performance optimisée (Redis, CDN, compression)

STANDARDS DE CODE
-----------------
- PHP: PSR-12, Laravel Pint
- JavaScript: ES6+, conventions camelCase
- CSS: Tailwind CSS, classes utilitaires
- Commentaires: Français, PHPDoc pour méthodes publiques
- Nommage: PascalCase (classes), camelCase (méthodes/variables)

TESTS
-----
- Tests unitaires (PHPUnit)
- Tests d'intégration
- Tests E2E (Laravel Dusk)
- Couverture de code > 80%

COMMANDES UTILES
----------------
# Génération du changelog
composer changelog

# Tests
php artisan test
composer test

# Développement
composer dev

# Optimisation
php artisan images:optimize
php artisan cache:warmup
